Jak vygenerovat token
1. Jdi na github.com/settings/tokens/new
2. Zvol Classic token, pojmenuj ho třeba slonek-web
3. Zaškrtni oprávnění repo (celé)
4. Klikni Generate token a zkopíruj ho sem

Token se uloží jen do tvého prohlížeče (localStorage). Nikdo jiný k němu nemá přístup.
